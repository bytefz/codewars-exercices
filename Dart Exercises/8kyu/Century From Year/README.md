# Century From Year

## Introduction
This first century spans from the *year 1*  up to _and including_ the year 100, the second century from the year 101 up to _and including_ the year 200, etc.

## Task
Given a year, return the century it is in.

## Examples
```
1705 --> 18
1900 --> 19
1601 --> 17
2000 --> 20
```
